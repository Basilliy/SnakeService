Для того что бы начать игру на компьютере должен быть открыть порт 55000 (как открыть порт - читай далее), а так же все игроки дожны быть в одной локальной сети.
Если компьютеры находятся далеко друг от друга или не в одной локалькой сети, то можно использовать программы для создания виртуальной локальной сети, такие как LogMein Hamachi.
При подключении к комнате в виртуальной локальной сети, если не удается подключится по адресу предлагаемому игрой, используйте адресс выдаваемый программой для создания виртуальной локальной сети.

Как открыть порт
Windows 7\8\8.1
Откройте Брандмауэр Windows.
Настройка встроенного брандмауэра находится в «Панели управления», раздел «Система и безопасность». В Windows 8 и 8.1 можно воспользоваться элементом рабочего стола — Поиск. Нужно начать вводить слово «Брандмауэр» и система выдаст ссылку в результатах. Ещё один быстрый и удобный способ быстро попасть в настройки Брандмауэра — нажать комбинацию клавиш Win+R: В строке открыть пишем firewall.cpl, нажимаем кнопку ОК.
В настройках встроенного брандмауэра  перейдите по ссылке «Дополнительные параметры».
Вам откроется раздел настройки Брандмауэра в режиме повышенной безопасности.
В поле слева кликаем на раздел «Правила для входящих соединений»: Откроется список всех правил для входящих соединений.
Теперь надо создать правило. Для этого справа, в окне «Действия», кликаем на ссылку «Создать правило».
Откроется Мастер создания правила для нового входящего подключения: Выбираем тип правила — «Для порта».
Нажимаем кнопку Далее. Теперь нужно выбрать тип протокола и указать порт.
«Тип протокола» для игр, как правило, «Протокол TCP».
В поле Определенные порты нужно указать номер нужного порта (в данном случае это 55000).
Нажимаем кнопку Далее. Ставим галку «Разрешить подключение». Нажимаем кнопку Далее.
В следующем окне ничего не меняем. Нажимаем кнопку Далее.
В поле Имя прописываем название для создаваемого правила — например Snake (имя может быть любым).
Нажимаем кнопку Готово.
После этого повторите процедуру, но уже выбрав в настройке Брандмауэра «Правила для исходящих соединений». В процедуре указать тот же порт (55000) и протокол (Протокол TCP).
Инструкция взята с http://nastroisam.ru/kak-otkryit-port-v-windows/
Там вы можете просмотреть скриншоты настройки порта.

Linux
Выполните команду:
iptables -I INPUT -p tcp -m tcp --dport 55000 -j ACCEPT
